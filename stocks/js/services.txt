Service Name:
	Page
Properties:
	title - title of the page
Methods:
	setTitle - set a new page title
	getTitle - get the current page title
	
========
	
Service Name:
	User
Properties:
	isLoggedIn - true if user is logged in
	jwt - JSON Web Token
	username - username
	avatar - filename of user's avatar image
	cart - array with user's shopping cart contents
Methods:
	register - sends username/password to server, if successful, registers account
	login - sends username/password to server, if successful, logs in
	logout - logs out, deletes local user info
	getUsername - get username
	getAvatar - get avatar image
